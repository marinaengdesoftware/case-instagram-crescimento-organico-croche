## Relação com Algoritmo

Quando um conteúdo recebe interações consistentes nos primeiros momentos após a publicação, o sistema de recomendação tende a expandir sua distribuição para novos grupos de usuários com comportamento semelhante.

Esse processo ocorre por meio da identificação de padrões de interesse e clusterização de audiência.
